# Purple Hat Institute — Website

Official website for **Purple Hat Institute**, an engineering coaching centre based in Nashik, Maharashtra, India.

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Home page — courses, testimonials, app download links |
| `about.html` | About the institute |
| `contact.html` | Contact details, map, and branch locations |

## Features

- Popular engineering courses with INR pricing
- Real student testimonials with photos
- Working Google Maps embed (KBT Circle, Nashik)
- Indira Nagar branch coming-soon modal
- Social links — YouTube, Instagram, LinkedIn
- Smooth page transition animations
- Mobile responsive

## Tech Stack

- HTML5 / CSS3 / Vanilla JS
- Bootstrap 5
- Font Awesome 6
- Google Fonts (Inter + Space Grotesk)

## Courses Offered

- Engineering Mathematics
- Basic Electrical Engineering
- Engineering Drawing
- Data Structures & Algorithms
- Engineering Physics
- Digital Electronics

## Locations

- **KBT Circle, Nashik** — Main branch
- **Indira Nagar, Nashik** — Coming soon (near Guru Gobind Singh College)
